- Ruby Install
  - https://github.com/rbenv/rbenv#installation
  - ```sudo apt install rbenv```
  - ```rbenv init```
  - ```rbenv install 3.1.2```
  - ```rbenv global 3.1.2```

  - bundle exec jekyll serve
